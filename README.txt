Ohjelma vaatii muutaman kolmannenosapuolen kirjaston

PIP TARVITAAN!!

Navigoi komentokehotteella tähän kansioon ja
Kirjoita komentokehotteeseen:

*
pip install -r needed.txt
*

Ja pip asentaa tarvitsemasi tiedostot ja paketit (3)
PyQt5, Tarvitaan GUI:n toimivuuteen
Beatifulsoup4, Käytetään nettisivujen parsimisessa
Requests, Käytetään uusien sivujen avaamiseen mahdollisimman nopeasti ja tehokkaasti

Käynnistä ohjelma (main.py), anna aloitus osoite HTTP/HTTPS muodossa sekä lukumäärä,
kuinka monta sivua haluat että käydään läpi.
Email osoitteet talleneetaan Libs\mails.txt tiedostoon
